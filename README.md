
## VSCode Info

Shortcuts
---

CTR + ~ -> Terminal

CTR + Tab -> Tab through open files

CMD + B -> Sidebar

CMD + , -> Settings

CMD + \ -> Open file in split window

CMD + W -> Close file

CMD + Shift + F -> Global searce

CMD + Shift + P -> Command pallete

CMD + D -> Select word

CMD + Shift + L -> Select all instances of a selected word
